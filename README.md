# trainingProject

This repository contains a one-page frontend portfolio for Zainab Alhommsi. The portfolio uses Bootstrap 5, Font Awesome, and custom JavaScript to provide a responsive dark/light theme experience with animated scroll sections.

## Included files
- `index.html` — main portfolio page
- `css/style.css` — custom portfolio styles and theme variables
- `javascript/script.js` — theme toggle, scroll animation, and contact form behavior
- `ZEANNAB ALHOMMSI.pdf` — resume source used for page content

## Notes
The page is built for a frontend developer role and is designed to be expanded with four additional color-themed pages in the future.
