# trainingProject

This repository contains a one-page frontend portfolio for me. The portfolio uses Bootstrap 5, Font Awesome, and custom JavaScript to provide a responsive dark/light theme experience with animated scroll sections.

## Included files
- `index.html` — main portfolio page
- `css/style.css` — custom portfolio styles and theme variables
- `javascript/script.js` — theme toggle, scroll animation, and contact form behavior

## other pages
- `blog.html` — blog page with articles and posts 
- `calculator.html` — simple calculator application
- `form.html` — contact form page with validation
- `store.html` — mock online store page with product listings and shopping cart functionality
